# How to open .md files with correct formatting

> *This file was written by Claude (Claude Code, claude.ai/code).*

`.md` files are Markdown files — plain text that looks much better when rendered.

---

## Easiest option (iPad & PC)
Go to **[dillinger.io](https://dillinger.io)** in any browser, then paste the file content in — it renders instantly. No install needed.

## On PC
Open with **Visual Studio Code** (free):
1. Open the file in VS Code
2. Press `Ctrl + Shift + V` to open the preview

## On iPad
Open the file in **Safari** — some `.md` files render directly.
Or use the free app **Markdown Viewer** from the App Store.

---

*If none of these work, the raw text is still fully readable — the formatting symbols (`**`, `#`, `-`) are minimal and easy to ignore.*
